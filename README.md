# QFT & Conversion from Classical to Quantum Circuits
Qiskit implementation of Quantum Fourier Transform (QFT) and conversion from classical to quantum circuits in the course [QPROG](https://utbildning.su.se/english/education/course-catalogue/ml/ml730n) at Department of Computer and System Sciences

## Authors
- Lars Herbold
- Fredrik Etsare

## Install
```bash
git clone https://github.com/fetsare/QFT-Conversion-from-Classical-to-Quantum-Circuits
cd QFT-Conversion-from-Classical-to-Quantum-Circuits
```

```bash
python3 -m venv venv
```

* On Linux/macOS:
```bash
source venv/bin/activate
```
* On Windows (PowerShell):

```powershell
.\venv\Scripts\Activate.ps1
```

```bash
pip install -r requirements.txt
```
